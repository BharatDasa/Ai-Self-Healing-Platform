# Scaling Guide

The platform uses AI metrics with KEDA autoscaling.

---

# Flow

```text
Kafka Events
      ↓
Self-Healing Engine
      ↓
predict_scale.py
      ↓
ai_target_replicas metric
      ↓
Prometheus
      ↓
KEDA
      ↓
Argo Rollout
      ↓
fraud-detection Pods
```

---

# Metrics

Exported metrics:

* ai_event_rate
* ai_model_score
* ai_target_replicas

---

# Decision Logic

NORMAL

Replica:

```text
1
```

ELEVATED

Replica:

```text
2
```

SCALE

Replica:

```text
3-5
```

---

# KEDA Configuration

```yaml
minReplicaCount: 1

maxReplicaCount: 5

pollingInterval: 15

cooldownPeriod: 60
```

---

# Prometheus Query

```promql
max(ai_target_replicas)
```

---

# Verify Scaling

```bash
kubectl get scaledobject -A

kubectl get hpa -A

kubectl get pods -n ai-platform
```

# Scaling Actions

KEDA scales:

fraud-detection Rollout
